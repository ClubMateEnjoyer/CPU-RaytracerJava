#CPU-RaytracerJava

Ein CPU-basierter Raytracer in Java, entwickelt im Rahmen des Moduls "Computergrafik Gundlagen".

Das Projekt unterstützt **Pathtracing**, **Phong- und Glass-Materialien**, **Texturen**, **OBJ-Modelle** und **BVH-Beschleunigungsstrukturen**.

## Features

-**Raytracing und Pathtracing**
  - Primärstrahlen, Sekundärstrahlen für Reflexion und Transparenz
  - Supersampling für Anti-Aliasing
-**Materialien**
  - Phongmaterial, Glassmaterial, Spiegelmaterial
-**Objekte**
  - Sphere, Plane, Triangle, Gruppen
  - import von OBJ-Dateien
-**Licht**
  - Direktionaless Licht, Punktlicht
-**Optimierung**
  -BVH für schnellere Intersektion
-**Output**
  - PNG
  - Anpassbare Auflösung durch Supersampling

## Voraussetzungen
  - Java 17 oder höher
  - Java compiler
